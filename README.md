# mern-auth-boilerplate

A full-stack authentication system built with **MongoDB, Express, React, and Node.js**. 
This project provides a robust foundation for user authentication using **JSON Web Tokens (JWT)** and password hashing with **Bcrypt**.

## Key Features
- **User Registration**: Secure sign-up with password encryption using Bcrypt.
- **User Login**: Identity verification and JWT issuance.
- **Persistent Auth**: Maintains user session using local storage or cookies.
- **Protected Routes**: Restricts access to specific pages for authenticated users only.
- **Logout**: Securely clears session/tokens on the client side.

## Tech Stack
### 1. Frontend
- **React.js**: Functional components and Hooks.
- **Axios**: Handling API requests.
- **React Router**: Client-side navigation and route protection.

### 2. Backend
- **Node.js & Express**: Server-side logic and RESTful API.
- **MongoDB & Mongoose**: NoSQL database and data modeling.
- **JSON Web Token (JWT)**: Secure data transmission between parties.
- **Bcrypt**: Password hashing for security.
